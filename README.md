## 📌 Project Overview

This is a full-stack clone of the Zerodha trading platform, built to simulate a modern stock trading interface. The application provides users with a clean dashboard, stock tracking UI, and portfolio management features.

---

## ✨ Features

- 🔐 User Authentication (Login/Register)
- 📊 Dashboard with stock overview
- 📈 Real-time-like stock UI (simulated data)
- 💼 Portfolio management
- 🛒 Buy/Sell interface (UI-based)
- 📱 Fully responsive design
